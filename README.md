# Stryde - Fitness Training App

A beginner-friendly web application for tracking workouts and fitness progress.

## Getting Started

### Prerequisites
- Node.js 16+ installed on your computer
- A code editor (VS Code recommended)
- Basic knowledge of JavaScript

### Installation

1. Clone this repository
```bash
git clone https://github.com/dakotataylor227-dotcom/Stryde.git
cd Stryde
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
